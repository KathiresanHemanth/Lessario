# Game Development - Learning Guide

## 🎯 Goal
Build games — from 2D mobile games to 3D AAA experiences. Game dev combines programming, art, physics, and storytelling.

## Core Concepts

1. **Game Loop** — Update → Render → Repeat, delta time, FPS
2. **2D Graphics** — Sprites, animation, tilemaps, pixel art
3. **Physics** — Collision detection, rigid body, forces, raycasting
4. **Input** — Keyboard, mouse, gamepad, touch
5. **Game Design** — Mechanics, level design, game feel, balancing
6. **3D Graphics** — Meshes, textures, lighting, shaders, cameras
7. **AI** — Pathfinding (A*), state machines, behavior trees, nav mesh
8. **Networking** — Multiplayer, client-server, P2P, lag compensation
9. **Audio** — Sound effects, music, 3D spatial audio
10. **Optimization** — LOD, occlusion culling, object pooling, profiling
11. **Shaders** — HLSL/GLSL, vertex/fragment shaders, post-processing

## Tips
- **Start with Unity + C#** — most jobs, best tutorials
- **Build complete games** — finish small games before attempting big ones
- **Game jams** — Ludum Dare, GMTK Game Jam (build game in 48 hours)
- **Publish on itch.io** — free, builds portfolio
- **Unreal for 3D** — if targeting AAA or cinematic quality
- **Godot is rising** — free, open source, great for learning

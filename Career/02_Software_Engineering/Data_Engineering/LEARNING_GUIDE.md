Build data pipelines that move, transform, and serve data at scale. One of the highest-paying and most in-demand roles in tech.

## Core Concepts

1. **SQL Mastery** — Joins, window functions, CTEs, query optimization
2. **Python for Data** — Pandas, PySpark, data manipulation
3. **ETL/ELT** — Extract, Transform, Load pipelines
4. **Data Warehouses** — Snowflake, BigQuery, Redshift, Databricks
5. **Streaming** — Apache Kafka, Flink, Spark Streaming
6. **Orchestration** — Apache Airflow, Dagster, Prefect
7. **Data Modeling** — Star schema, snowflake schema, dimensional modeling
8. **Cloud Data** — AWS Glue, S3, Athena, GCP Dataflow
9. **Data Quality** — Great Expectations, dbt tests, validation
10. **dbt (Data Build Tool)** — SQL transformations, version-controlled models

## The Modern Data Stack

```
Sources → Ingestion → Storage → Transform → Serve
           (Fivetran)   (S3/GCS)  (dbt/Spark)  (Snowflake)
                              ↓
                        Orchestration (Airflow)
                              ↓
                        Visualization (Looker/Metabase)
```

## Tips
- **SQL is king** — master it before anything else
- **Learn Airflow** — most-used orchestration tool
- **dbt is essential** — transforms how teams do analytics
- **Spark for big data** — when Pandas isn't enough
- **Kafka for streaming** — real-time data is the future

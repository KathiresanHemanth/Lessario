# C Programming - Project-Based Learning Roadmap for Zoho

## Level 1: Basics (Variables, I/O, Control Flow)

- Build a simple calculator that performs +, -, *, / operations
- **Concepts**: Variables, printf/scanf, if-else, functions
- **Difficulty**: ⭐

### Project 2: Grade Management System
- Take marks as input, calculate average, display grade (A/B/C/D/F)

### Project 3: Number Guessing Game
- Computer picks random number, user guesses it

### Project 4: Multiplication Table Generator
- Display multiplication tables based on user input
- **Concepts**: nested loops, formatting output

## Recommended Learning Path

**Week 1-2**: Projects 1-4 (Basics)
**Week 3-4**: Projects 5-8 (Arrays & Strings)
**Week 5-6**: Projects 9-11 (Pointers)
**Week 7-8**: Projects 12-15 (Data Structures)
**Week 9-10**: Projects 16-19 (File Handling)
**Week 11-12**: Projects 20-24 (Advanced - Interview prep)

## Tips for Success
1. Start with projects you can complete in 1-2 hours
2. Don't just copy code - type everything manually
3. Understand every line of code you write
4. Test edge cases (empty inputs, boundary conditions)
5. Gradually increase complexity
6. Practice debugging with print statements
7. Comment your code properly
8. Focus on clean, readable code

## Key Topics to Master Before Zoho Interview
- Pointers and memory management
- Linked lists and other data structures
- String manipulation
- File I/O
- Algorithm complexity (Big O)
- Recursion
- Problem-solving skills

CALCULATOR PROJECT - STEP BY STEP TASKS

=== TASK 1: Print Welcome Message ===
Create a file called Calculator_Step1.c with this code:

#include <stdio.h>

int main() {
    printf("Welcome to Calculator!\n");
    system("pause");
    return 0;
}

What to do:
1. Create new file Calculator_Step1.c
2. Copy the code above
3. Compile: gcc Calculator_Step1.c -o calc_step1.exe
4. Run: .\calc_step1.exe
5. You should see "Welcome to Calculator!"

=== TASK 8: Combine Everything - Final Calculator ===
Create Calculator_Final.c

#include <stdio.h>

int main() {
    int num1, num2, result;
    char operation;
    
    printf("=== Simple Calculator ===\n");
    printf("Enter first number: ");
    scanf("%d", &num1);
    
    printf("Enter operator (+, -, *, /): ");
    scanf(" %c", &operation);
    
    printf("Enter second number: ");
    scanf("%d", &num2);
    
    if(operation == '+') {
        result = num1 + num2;
        printf("%d + %d = %d\n", num1, num2, result);
    } 
    else if(operation == '-') {
        result = num1 - num2;
        printf("%d - %d = %d\n", num1, num2, result);
    }
    else if(operation == '*') {
        result = num1 * num2;
        printf("%d * %d = %d\n", num1, num2, result);
    }
    else if(operation == '/') {
        if(num2 != 0) {
            result = num1 / num2;
            printf("%d / %d = %d\n", num1, num2, result);
        } else {
            printf("ERROR: Cannot divide by zero!\n");
        }
    }
    else {
        printf("Invalid operator!\n");
    }
    
    system("pause");
    return 0;
}

What to do:
1. Create file Calculator_Final.c
2. Copy code
3. Compile and run
4. Test all operations:
   - Enter: 10, +, 5 → should print "10 + 5 = 15"
   - Enter: 20, -, 8 → should print "20 - 8 = 12"
   - Enter: 6, *, 7 → should print "6 * 7 = 42"
   - Enter: 20, /, 4 → should print "20 / 4 = 5"
   - Enter: 10, /, 0 → should print error
   - Enter: 5, %, 2 → should print "Invalid operator!"

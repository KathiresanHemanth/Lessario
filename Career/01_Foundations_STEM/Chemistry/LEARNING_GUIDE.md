## Learning Path

### Phase 1: Fundamentals (Week 1-4)
1. **Atomic Structure** — Electrons, orbitals, periodic table, electron configuration
2. **Chemical Bonding** — Ionic, covalent, metallic, hydrogen, van der Waals
3. **States of Matter** — Solids, liquids, gases, phase diagrams, intermolecular forces
4. **Stoichiometry** — Moles, balancing equations, limiting reagent, yield
5. **Solutions** — Concentration, molarity, solubility, dilution

### Phase 2: Reactions & Energy (Week 5-8)
6. **Chemical Reactions** — Types (synthesis, decomposition, redox, acid-base)
7. **Thermochemistry** — Enthalpy, Hess's law, calorimetry, exo/endothermic
8. **Kinetics** — Rate laws, activation energy, catalysts, Arrhenius equation
9. **Equilibrium** — Le Chatelier's principle, Kp, Kc, equilibrium calculations
10. **Acids & Bases** — pH, pKa, buffers, titration, Henderson-Hasselbalch

### Phase 3: Applied Chemistry (Week 9-14)
11. **Electrochemistry** — Galvanic cells, electrolysis, Nernst equation, batteries
12. **Organic Chemistry** — Hydrocarbons, functional groups, polymers, reactions
13. **Materials Science** — Crystal structures, alloys, polymers, ceramics, composites
14. **Nuclear Chemistry** — Radioactivity, fission, fusion, half-life
15. **Surface Chemistry** — Catalysis, adsorption, colloids, nanoparticles

### Phase 4: Industry Chemistry (Week 15-20)
16. **Polymer Chemistry** — Plastics, rubber, resins, 3D printing materials
17. **Semiconductor Chemistry** — Silicon purification, CVD, doping, photoresists
18. **Battery Chemistry** — Li-ion, LFP, NMC, solid electrolytes, degradation
19. **Propellant Chemistry** — RP-1/LOX, hydrazine, solid propellants, Isp
20. **Green Chemistry** — Sustainability, hydrogen production, CO₂ capture

## Key Equations
```
PV = nRT                          (Ideal Gas Law)
ΔG = ΔH - TΔS                    (Gibbs Free Energy)
E = E° - (RT/nF)ln(Q)            (Nernst Equation)
k = Ae^(-Ea/RT)                   (Arrhenius Equation)
pH = -log[H⁺]                    (pH calculation)
rate = k[A]^m[B]^n                (Rate Law)
```

## Chemistry in Elon Musk's Companies

### Tesla — Battery Chemistry
```
Li-ion cell: LiCoO₂ cathode → Li⁺ ions → Graphite anode
Charging:    LiCoO₂ → Li⁺ + e⁻ + CoO₂ (cathode releases Li⁺)
Discharging: Reverse reaction, electrons flow through circuit

Tesla 4680 cell innovations:
- Cathode: NMC (Nickel Manganese Cobalt) → higher energy
- Anode: Silicon-graphite → more Li⁺ storage
- Electrolyte: Dry electrode process → cheaper manufacturing
- Goal: Reduce $/kWh by 50%
```

### SpaceX — Rocket Propellant Chemistry
```
Raptor Engine: CH₄ + 2O₂ → CO₂ + 2H₂O + ENERGY
  Methane (CH₄) + Liquid Oxygen (LOX)
  Why methane? Can be manufactured on Mars from CO₂ + H₂O

Merlin Engine: RP-1 + O₂ → CO₂ + H₂O
  RP-1 = refined kerosene
  
Specific Impulse (Isp) depends on:
  - Exhaust molecular weight (lighter = better)
  - Combustion temperature (hotter = better)
  - Chamber pressure
```

### Semiconductor — Silicon Chemistry
```
Sand (SiO₂) → Purify to 99.9999999% pure Si
SiO₂ + C → Si + CO₂  (at 1800°C)
Then: Czochralski process → grow single crystal ingot
Then: Wafer slicing → polishing → photolithography
Doping: Add P (n-type) or B (p-type) to control conductivity
```

# Mathematics - STEM Foundation Guide

## 🎯 Goal
Build the math foundation required for ALL engineering and tech fields. Math is the language of the universe — every discipline depends on it.

## Learning Path (in order)

### Phase 1: Absolute Basics (Week 1-2)
1. **Number Systems** — Natural, Integer, Rational, Real, Complex
2. **Arithmetic** — BODMAS, fractions, percentages, ratios
3. **Algebra** — Variables, equations, inequalities, polynomials
4. **Functions** — Linear, quadratic, exponential, logarithmic

### Phase 2: Core Math (Week 3-6)
5. **Trigonometry** — sin/cos/tan, unit circle, identities, inverse
6. **Coordinate Geometry** — Lines, circles, conic sections
7. **Sequences & Series** — AP, GP, convergence, sigma notation
8. **Matrices & Determinants** — Operations, inverse, eigenvalues

### Phase 3: Advanced (Week 7-12)
9. **Calculus** — Limits, derivatives, integrals, chain rule
10. **Multivariable Calculus** — Partial derivatives, gradient, Jacobian
11. **Differential Equations** — ODE, PDE, Laplace transforms
12. **Linear Algebra** — Vector spaces, transformations, SVD, PCA

### Phase 4: Applied (Week 13-16)
13. **Probability** — Bayes, distributions, expectation, variance
14. **Statistics** — Hypothesis testing, regression, confidence intervals
15. **Discrete Math** — Logic, sets, graphs, combinatorics, number theory
16. **Optimization** — Gradient descent, convex optimization, LP

## Quick Reference

### Key Formulas
```
Quadratic: x = (-b ± √(b²-4ac)) / 2a
Distance:  d = √((x₂-x₁)² + (y₂-y₁)²)
Derivative: d/dx(xⁿ) = n·xⁿ⁻¹
Integral:   ∫xⁿ dx = xⁿ⁺¹/(n+1) + C
Euler:      e^(iπ) + 1 = 0
Bayes:      P(A|B) = P(B|A)·P(A) / P(B)
```

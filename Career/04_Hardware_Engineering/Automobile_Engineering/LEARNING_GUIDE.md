# Automobile Engineering - Learning Guide

## 🎯 Goal
Understand vehicle systems from engines to electronics — covering mechanical, electrical, and software aspects of modern automobiles.

---

## Prerequisites
✅ Physics — Thermodynamics, Mechanics, Fluid Dynamics
✅ Mathematics — Calculus, Differential Equations
✅ Programming — C/C++ (ECU), MATLAB (simulation), Python
✅ Materials Science — Metals, composites, polymers

---

## Core Concepts (in order)

1. **Vehicle Architecture** — Chassis, body, powertrain layout
2. **Internal Combustion Engines** — 4-stroke, diesel, petrol, compression
3. **Thermodynamics** — Otto cycle, Diesel cycle, efficiency
4. **Drivetrain** — Clutch, gearbox, differential, axles, FWD/RWD/AWD
5. **Suspension & Steering** — MacPherson, double wishbone, rack & pinion
6. **Braking Systems** — Disc, drum, ABS, regenerative braking
7. **Electrical Systems** — Battery, alternator, starter, wiring harness
8. **Vehicle Dynamics** — Handling, stability, traction, tire mechanics
9. **Automotive Electronics (ECU)** — Engine management, sensors, CAN bus
10. **Electric Vehicles** — Motors, battery packs, BMS, charging
11. **ADAS & Autonomous Vehicles** — Sensors, perception, decision making

---

## Key Formulas

### Engine Performance
```
Power (HP) = Torque (Nm) × RPM / 5252
Thermal Efficiency = Work Output / Heat Input
Compression Ratio = V_max / V_min
```

### Vehicle Dynamics
```
Traction Force = μ × Normal Force
Braking Distance = v² / (2 × μ × g)
Cornering Force = m × v² / r
```

### Electric Vehicle
```
Range = Battery Capacity (kWh) / Energy Consumption (kWh/km)
Motor Torque = (P × 60) / (2π × N)
```

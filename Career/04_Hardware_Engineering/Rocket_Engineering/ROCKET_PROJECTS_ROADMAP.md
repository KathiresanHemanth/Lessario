# Rocket Engineering - Project-Based Learning Roadmap

## Level 1: Fundamentals (Weeks 1-3)

### Project 1: Rocket Equation Calculator
- Calculate delta-v, mass ratio, specific impulse for different rockets
- **Concepts**: Tsiolkovsky equation, mass ratio, Isp
- **Difficulty**: ⭐

### Project 2: Trajectory Simulator (1D)
- Simulate vertical rocket flight — thrust, gravity, drag
- **Concepts**: Newton's laws, numerical integration, drag
- **Difficulty**: ⭐⭐

### Project 3: Nozzle Design Calculator
- Calculate throat area, exit area, expansion ratio for convergent-divergent nozzle
- **Concepts**: Isentropic flow, Mach number, choked flow
- **Difficulty**: ⭐⭐

### Project 4: Propellant Comparison Tool
- Compare solid, liquid, and hybrid propellants by Isp, density, cost
- **Concepts**: Propellant chemistry, performance parameters
- **Difficulty**: ⭐

---

## Level 2: Orbital Mechanics (Weeks 4-7)

### Project 5: Orbital Simulator (2D)
- Simulate satellite orbits around Earth using numerical methods
- **Concepts**: Kepler's laws, gravitational force, RK4 integration
- **Difficulty**: ⭐⭐⭐

### Project 6: Hohmann Transfer Calculator
- Calculate delta-v for orbit transfers between circular orbits
- **Concepts**: Hohmann transfer, orbital energy, vis-viva equation
- **Difficulty**: ⭐⭐

### Project 7: Launch Window Calculator
- Determine optimal launch windows for interplanetary missions
- **Concepts**: Planetary alignment, synodic period, porkchop plots
- **Difficulty**: ⭐⭐⭐

### Project 8: Multi-Stage Rocket Optimizer
- Optimize stage mass ratios for maximum payload
- **Concepts**: Staging theory, Lagrange multipliers, optimization
- **Difficulty**: ⭐⭐⭐

---

## Level 3: Design & Simulation (Weeks 8-12)

### Project 9: Model Rocket Design (OpenRocket)
- Design a model rocket, simulate flight, predict altitude
- **Concepts**: OpenRocket, stability, center of pressure
- **Difficulty**: ⭐⭐

### Project 10: Rocket Structure FEA
- Stress analysis of rocket body tube under thrust loads
- **Concepts**: FEA, buckling, thin-wall pressure vessels
- **Difficulty**: ⭐⭐⭐

### Project 11: Attitude Control Simulator
- Simulate reaction control system (RCS) for attitude adjustment
- **Concepts**: Euler angles, moment of inertia, thruster control
- **Difficulty**: ⭐⭐⭐⭐

### Project 12: Heat Shield Analysis
- Calculate re-entry heating and thermal protection requirements
- **Concepts**: Stagnation temperature, ablation, heat flux
- **Difficulty**: ⭐⭐⭐⭐

---

## Level 4: Advanced (Weeks 13-18)

### Project 13: Full Mission Simulator
- End-to-end mission: launch → orbit insertion → transfer → landing
- **Concepts**: Mission design, delta-v budget, timeline
- **Difficulty**: ⭐⭐⭐⭐

### Project 14: Liquid Engine Cycle Analysis
- Model gas generator / staged combustion engine cycles
- **Concepts**: Turbopump, preburner, cycle efficiency
- **Difficulty**: ⭐⭐⭐⭐

### Project 15: Landing Simulation (Powered Landing)
- Simulate SpaceX-style powered landing with thrust control
- **Concepts**: Hover slam, guidance algorithms, control
- **Difficulty**: ⭐⭐⭐⭐

### Project 16: Satellite Constellation Design
- Design a LEO constellation for communication coverage
- **Concepts**: Walker constellation, coverage analysis, ground track
- **Difficulty**: ⭐⭐⭐⭐

---

## Recommended Learning Path

**Week 1-3**: Projects 1-4 (Fundamentals)
**Week 4-7**: Projects 5-8 (Orbital Mechanics)
**Week 8-12**: Projects 9-12 (Design & Simulation)
**Week 13-18**: Projects 13-16 (Advanced)

## Tips for Success
1. Start with OpenRocket — free and great for learning
2. Master MATLAB/Python for trajectory calculations
3. Study real missions (Apollo, Chandrayaan, Falcon 9)
4. Learn orbital mechanics deeply — it's the heart of rocketry
5. Build model rockets if possible (Estes kits)
6. Follow ISRO recruitment for career opportunities

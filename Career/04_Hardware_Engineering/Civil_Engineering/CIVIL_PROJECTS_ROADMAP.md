# Civil Engineering - Project-Based Learning Roadmap

## Level 1: Fundamentals (Weeks 1-3)

### Project 1: Beam Load Calculator
- Calculate reactions, shear force, bending moment for simply supported beams
- **Concepts**: Statics, equilibrium, SFD/BMD
- **Difficulty**: ⭐

### Project 2: Truss Analyzer
- Analyze forces in a simple truss (method of joints/sections)
- **Concepts**: Truss analysis, tension/compression
- **Difficulty**: ⭐⭐

### Project 3: Concrete Mix Design Calculator
- Calculate mix proportions for different grades (M20, M25, M30)
- **Concepts**: IS 10262, water-cement ratio, aggregates
- **Difficulty**: ⭐⭐

### Project 4: Soil Classification Tool
- Classify soil based on grain size distribution and Atterberg limits
- **Concepts**: IS soil classification, plasticity chart
- **Difficulty**: ⭐⭐

---

## Level 2: CAD & Design (Weeks 4-7)

### Project 5: 2D House Plan (AutoCAD)
- Draw a complete 2D floor plan with dimensions
- **Concepts**: AutoCAD commands, layers, dimensioning
- **Difficulty**: ⭐⭐

### Project 6: RCC Column Design
- Design a short column as per IS 456
- **Concepts**: Axial load, reinforcement, IS code provisions
- **Difficulty**: ⭐⭐⭐

### Project 7: Steel Connection Design
- Design bolted and welded connections per IS 800
- **Concepts**: Bolt strength, weld size, connection types
- **Difficulty**: ⭐⭐⭐

### Project 8: Retaining Wall Design
- Design a cantilever retaining wall with stability checks
- **Concepts**: Earth pressure, overturning, sliding, bearing
- **Difficulty**: ⭐⭐⭐

---

## Level 3: Software & Analysis (Weeks 8-11)

### Project 9: STAAD Pro — Simple Frame
- Model and analyze a 2D portal frame in STAAD Pro
- **Concepts**: STAAD modeling, loading, results interpretation
- **Difficulty**: ⭐⭐⭐

### Project 10: Building Estimation & Costing
- Prepare detailed estimate for a small residential building
- **Concepts**: Quantity takeoff, rates, BOQ, abstract
- **Difficulty**: ⭐⭐⭐

### Project 11: Road Geometric Design
- Design horizontal and vertical curves for a highway
- **Concepts**: IRC codes, sight distance, superelevation
- **Difficulty**: ⭐⭐⭐

### Project 12: Water Supply Network Design
- Design pipe network for a small colony
- **Concepts**: Hardy Cross method, pipe sizing, pressure
- **Difficulty**: ⭐⭐⭐

---

## Level 4: Advanced Projects (Weeks 12-16)

### Project 13: Earthquake Resistant Building Design
- Design a multi-story RCC building for seismic loads
- **Concepts**: IS 1893, response spectrum, ductile detailing
- **Difficulty**: ⭐⭐⭐⭐

### Project 14: Bridge Design
- Design a simple RCC slab bridge
- **Concepts**: IRC loading, deck slab, abutment design
- **Difficulty**: ⭐⭐⭐⭐

### Project 15: Construction Project Schedule
- Create CPM/PERT network, Gantt chart for building construction
- **Concepts**: Project management, critical path, float
- **Difficulty**: ⭐⭐⭐

### Project 16: BIM Model (Revit)
- Create a 3D BIM model of a residential building
- **Concepts**: Revit, BIM workflow, clash detection
- **Difficulty**: ⭐⭐⭐⭐

---

## Recommended Learning Path

**Week 1-3**: Projects 1-4 (Fundamentals)
**Week 4-7**: Projects 5-8 (CAD & Design)
**Week 8-11**: Projects 9-12 (Software & Analysis)
**Week 12-16**: Projects 13-16 (Advanced)

## Tips for Success
1. Master AutoCAD — it's used everywhere in civil
2. Learn IS codes — essential for Indian practice
3. Visit construction sites regularly
4. Practice quantity estimation by hand
5. Learn STAAD Pro or ETABS for structural design
6. GATE exam preparation opens many PSU doors

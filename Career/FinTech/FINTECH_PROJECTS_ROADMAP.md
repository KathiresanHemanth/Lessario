# FinTech - Project-Based Learning Roadmap

## Level 1: Finance & Data Basics (Weeks 1-3)

### Project 1: Personal Finance Tracker
- Build a CLI app to track income, expenses, and savings
- **Concepts**: Python, file I/O, data structures, categories
- **Difficulty**: ⭐

### Project 2: Stock Data Analyzer
- Fetch stock data, calculate returns, moving averages, plot charts
- **Concepts**: yfinance, Pandas, Matplotlib, financial metrics
- **Difficulty**: ⭐⭐

### Project 3: Loan EMI Calculator
- Calculate EMI, total interest, amortization schedule
- **Concepts**: Compound interest, financial formulas, tables
- **Difficulty**: ⭐

### Project 4: Currency Converter API
- Build a REST API that converts currencies using live rates
- **Concepts**: FastAPI/Flask, external APIs, JSON, exchange rates
- **Difficulty**: ⭐⭐

## Tips for Success
1. Build Razorpay/Stripe integration — most asked in interviews
2. Learn SQL deeply — FinTech runs on databases
3. Understand UPI architecture — India's #1 payment system
4. Security is non-negotiable — learn PCI-DSS basics
5. Study RBI guidelines for compliance knowledge
6. Build a portfolio with working payment integrations
7. Contribute to open-source FinTech projects

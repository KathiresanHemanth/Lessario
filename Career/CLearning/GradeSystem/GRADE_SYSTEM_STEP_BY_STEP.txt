#include <stdio.h>

int main() {
    int marks[5];

CODE EXPLANATION:
- int marks[5] → array that stores 5 integers
  - marks[0] = first subject
  - marks[1] = second subject
  - marks[4] = fifth subject

- for(int i = 0; i < 5; i++) → loop that runs 5 times
  - i starts at 0
  - loop continues while i < 5
  - after each loop, i increases by 1

- marks[i] → accesses the i-th element in array
- i + 1 → adds 1 to i (so we show "Subject 1" instead of "Subject 0")

=== TASK 4: Calculate Total Marks ===
Create GradeSystem_Step4.c

#include <stdio.h>

int main() {
    int marks[5];
    float total = 0;
    
    printf("=== Grade Management System ===\n");
    printf("Enter marks for 5 subjects (0-100):\n");
    
    for(int i = 0; i < 5; i++) {
        printf("Subject %d: ", i + 1);
        scanf("%d", &marks[i]);
        total += marks[i];  // ADD each mark to total
    }
    
    printf("\n--- Results ---\n");
    printf("Total Marks: %.0f\n", total);
    
    system("pause");
    return 0;
}

CODE EXPLANATION:
- float total = 0 → variable to store total (decimal numbers)
  - we use float because average might have decimals
  - = 0 means we start at 0

- total += marks[i] → means: total = total + marks[i]
  - += is shorthand for "add and store"
  - so if total was 80 and marks[i] is 90, total becomes 170

- %.0f → format for printing decimals
  - % = special format code
  - .0 = show 0 decimal places
  - f = means it's a float number

int main() {
    int marks[5];
    float total = 0, average;

}

CODE EXPLANATION:
- average = total / 5 → divides total by 5 subjects to get average
  - if total is 438, average = 438 / 5 = 87.6

- float average → stores decimal result
  - if we used int, it would show 87 (without decimals)

- %.2f → format for printing decimals
  - .2 = show 2 decimal places
  - so 87.6 displays as 87.60

=== TASK 6: Assign Grade Based on Average ===
Create GradeSystem_Step6.c

#include <stdio.h>

int main() {
    int marks[5];
    float total = 0, average;
    char grade;
    
    printf("=== Grade Management System ===\n");
    printf("Enter marks for 5 subjects (0-100):\n");
    
    for(int i = 0; i < 5; i++) {
        printf("Subject %d: ", i + 1);
        scanf("%d", &marks[i]);
        total += marks[i];
    }
    
    average = total / 5;
    
    // DETERMINE GRADE
    if(average >= 90) {
        grade = 'A';
    } 
    else if(average >= 80) {
        grade = 'B';
    } 
    else if(average >= 70) {
        grade = 'C';
    } 
    else if(average >= 60) {
        grade = 'D';
    } 
    else {
        grade = 'F';
    }
    
    printf("\n--- Results ---\n");
    printf("Total Marks: %.0f\n", total);
    printf("Average: %.2f\n", average);
    printf("Grade: %c\n", grade);
    
    system("pause");
    return 0;
}

CODE EXPLANATION:
- char grade → variable to store a single character (letter A, B, C, D, or F)

- if(average >= 90) → checks if average is greater than or equal to 90
  - >= means "greater than or equal to"

- if-else chain → checks multiple conditions:
  - FIRST if average >= 90 → grade = 'A'
  - ELSE if average >= 80 → grade = 'B'
  - ELSE if average >= 70 → grade = 'C'
  - ELSE if average >= 60 → grade = 'D'
  - ELSE → grade = 'F' (all other cases)

- 'A' is a CHAR (single character in quotes)
  - different from "A" which is a STRING (group of characters)

- %c → prints a single character

GRADE SCALE:
- 90-100 = A (Excellent)
- 80-89  = B (Good)
- 70-79  = C (Average)
- 60-69  = D (Below Average)
- Below 60 = F (Fail)

=== TASK 7: FINAL PROJECT - Complete Grade System ===
Create GradeSystem_Final.c

#include <stdio.h>

int main() {
    int marks[5];
    float total = 0, average;
    char grade;
    
    printf("=== Grade Management System ===\n");
    printf("Enter marks for 5 subjects (0-100):\n");
    
    // INPUT MARKS FOR 5 SUBJECTS
    for(int i = 0; i < 5; i++) {
        printf("Subject %d: ", i + 1);
        scanf("%d", &marks[i]);
        total += marks[i];
    }
    
    // CALCULATE AVERAGE
    average = total / 5;
    
    // DETERMINE GRADE
    if(average >= 90) {
        grade = 'A';
    } 
    else if(average >= 80) {
        grade = 'B';
    } 
    else if(average >= 70) {
        grade = 'C';
    } 
    else if(average >= 60) {
        grade = 'D';
    } 
    else {
        grade = 'F';
    }
    
    // DISPLAY RESULTS
    printf("\n--- Results ---\n");
    printf("Total Marks: %.0f\n", total);
    printf("Average: %.2f\n", average);
    printf("Grade: %c\n", grade);
    
    system("pause");
    return 0;
}

CODE EXPLANATION - COMPLETE PROGRAM:
1. Include stdio.h library
2. Declare variables:
   - marks[5] → array for 5 subjects
   - total → sum of all marks
   - average → marks divided by 5
   - grade → letter grade (A/B/C/D/F)

3. Input Phase:
   - Loop 5 times to get marks for each subject
   - Add each mark to total

4. Calculate Phase:
   - Divide total by 5 to get average

5. Grade Assignment Phase:

6. Output Phase:

## Core Concepts

1. **Networking** — TCP/IP, DNS, HTTP/S, ports, protocols, Wireshark
2. **Linux** — Command line, permissions, scripting, services
3. **Web Security** — OWASP Top 10, XSS, SQLi, CSRF, SSRF
4. **Cryptography** — AES, RSA, hashing, TLS, PKI, digital signatures
5. **Network Security** — Firewalls, IDS/IPS, VPN, sniffing, spoofing
6. **Penetration Testing** — Recon, scanning, exploitation, post-exploitation
7. **Reverse Engineering** — Assembly, Ghidra, IDA Pro, malware analysis
8. **Forensics** — Disk, memory, network forensics, incident response
9. **Cloud Security** — AWS/Azure security, IAM, misconfigurations
10. **Security Operations** — SIEM, SOC, threat hunting, log analysis

## Tips
- **Start with TryHackMe** — beginner-friendly, guided
- **Move to Hack The Box** — real-world scenarios
- **Learn Linux deeply** — every tool runs on it
- **Bug bounties** = real income while learning (HackerOne, Bugcrowd)
- **OSCP** is the most respected cert for pentesting jobs

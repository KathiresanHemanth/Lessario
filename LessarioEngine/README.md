# 🔱 Lessario Engine v0.1.0

A high-performance C++ game engine optimized for the **Intel Core i3-2100** and **NVIDIA GT 710**.

## 💻 Targeted Hardware Profile
- **Architecture**: x64
- **Graphics API**: OpenGL 3.3 Core Profile
- **Optimal Resolution**: 1280x720 (720p)
- **Feature Set**: VSync Enabled, Low-latency Game Loop

## 🛠️ Build Instructions
1. **Requirements**: 
   - Visual Studio 2026 Community (with C++ Desktop Workload)
   - CMake 3.20+
   - SDL2
2. **Setup**:
   - `mkdir build`
   - `cd build`
   - `cmake ..`
3. **Run**: 
   - Open `LessarioEngine.sln` and press **F5**.

## 📂 Structure
- `src/`: Core logic and Engine Heartbeat.
- `include/`: Public headers.
- `assets/`: Shaders and textures (Copy to build dir on build).

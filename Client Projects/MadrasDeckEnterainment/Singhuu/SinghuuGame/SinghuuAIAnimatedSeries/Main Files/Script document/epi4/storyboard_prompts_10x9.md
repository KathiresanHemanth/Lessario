# Episode 4: 10-Prompt Storyboard (3x3 Grid Format)

This document provides **10 master prompts** to generate storyboard sheets. Each prompt is designed for a **3x3 grid** (9 shots per prompt), covering the entire 22-minute episode of *Singu - The Raja*.

---

**Prompt:** [Masterpiece, 8k, Ultra-detailed] A professional 3x3 storyboard grid sheet, high-end 3D stylized cinematic animation, Octane render.

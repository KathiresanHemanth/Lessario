# Gameplay Mechanics: The Council Swap System

> *Core mechanic for "Nama Thala Singhuu: Forest Run"*

---

## 1. The Dynamic Swap (Runner Core Mechanic)

The **character swap** is what differentiates Singhuu from every other runner on the market. It transforms a simple "run and dodge" game into a strategic "run, assess, and swap" experience.

### How It Works
- **Active Party:** The player takes 3 characters into each run (selected pre-run).
- **Swap Input:** Tap the character portrait at the bottom of the screen (or swipe with two fingers) to cycle characters instantly.
- **Swap Animation:** A 0.3-second flourish effect (character slides in from the side) — gameplay does NOT pause during swap.
- **Cooldown:** 2-second cooldown between swaps (prevents spam, adds strategic timing).

### Swap Strategy Examples
| Obstacle Ahead | Ideal Character | Why |
|----------------|----------------|-----|
| Wide gap | Thanthiraa (Fox) | Dash Burst covers the distance |
| Low hanging branches | Sssaarapaambu (Snake) | Tunnel Slide goes under them |
| Charging buffalo | Singu (Lion) | Royal Roar clears the path |
| Vertical cliff | Kongini (Gorilla) | Power Climb scales it instantly |
| Coin cluster in the air | Meenukutti (Bird) | Sky Glide reaches aerial pickups |
| Incoming boulder | Suttapazham (Camel) | Endurance Shield absorbs the hit |

## 2. Character Ability Details

### Singu — The Raja 🦁
- **Active: Royal Roar** — Press ability button → shockwave shoots forward, destroying obstacles in a 5-meter cone for 2 seconds.
- **Passive: Leader's Aura** — +10% coin pickup radius while Singu is active.
- **Cooldown:** 8 seconds.

### Thanthiraa — The Fox 🦊
- **Active: Dash Burst** — Instantly teleport forward 10 meters, invincible during dash.
- **Passive: Sly Feet** — Slightly faster base run speed than other characters.
- **Cooldown:** 6 seconds.

### Meenukutti — The Bird 🐦
- **Active: Sky Glide** — Lift off the ground for 3 seconds, floating above all ground-level obstacles.
- **Passive: Keen Eye** — Collectibles glow brighter and are visible from further away.
- **Cooldown:** 7 seconds.

### Kongini — The Scholar 🦍 *(Phase 2)*
- **Active: Power Climb** — When hitting a wall or cliff, auto-climb to the top instead of dying.
- **Passive: Pattern Sense** — Obstacles flash a 0.5-second warning indicator before appearing.
- **Cooldown:** 10 seconds.

### Sssaarapaambu — The Snake 🐍 *(Phase 2)*
- **Active: Tunnel Slide** — Flatten and slide through any low gap or narrow passage for 2 seconds.
- **Passive: Second Chance** — First death per run is survived with 1 HP (once per run).
- **Cooldown:** 6 seconds.

### Suttapazham — The Camel 🐪 *(Phase 2)*
- **Active: Endurance Shield** — Block the next incoming hit completely (any obstacle).
- **Passive: Cargo Hump** — 2x coin magnet range while active.
- **Cooldown:** 12 seconds.

### Muttagose — The Hare 🐇 *(Phase 2)*
- **Active: Double Jump** — Perform a second jump mid-air, reaching platforms otherwise impossible.
- **Passive: Speed Bunny** — Score multiplier increases 10% faster than other characters.
- **Cooldown:** 5 seconds.

## 4. Scoring System

| Action | Points |
|--------|--------|
| Distance run (per 10m) | +10 |
| Forest Coin collected | +5 |
| Rare Herb collected (Ep 2 themed) | +25 |
| Obstacle dodged (close miss) | +15 (Near Miss Bonus!) |
| Successful ability use | +20 |
| Character swap at the right moment | +30 (Perfect Swap Bonus!) |
| Mini-game completion | +100–500 |

### Combo System
- Chaining successful dodges and ability uses without getting hit builds a **combo multiplier** (up to 5x).
- The combo resets on hit or if 5 seconds pass without any action.

## 5. Progression System

### Character Upgrades
- Spend coins to level up characters (Level 1–10).
- Each level improves: ability duration (+0.2s), cooldown reduction (-0.5s), passive strength (+5%).

### Milestones (Achievement System)
| Milestone | Requirement | Reward |
|-----------|-------------|--------|
| First Run | Complete your first run | 100 Coins |
| Council of Three | Unlock 3 characters | Exclusive skin for Singu |
| Full Council | Unlock all 7 characters | "Raja's Crown" cosmetic |
| Marathon Runner | Run 10,000m in a single run | Legendary environment skin |
| Perfect Swapper | Get 50 "Perfect Swap Bonus" in one run | Title: "Swap Master" |

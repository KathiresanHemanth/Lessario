“My dear Chutti Friends...
Inniku kaattula ellarum bayandhuttaanga. Thanniye illa nu panic
aayittaanga.
Neengalum apdi thaan nenachingalaa?
Small problem vandhaalum... udane panic aaidaathinga?

Namma brain romba powerful.
Aana panic pannumbodhu... adhu work aagave aagathu.
Naan enna panninen theriyuma?
Konjam amaidhiyaa yosichen.
Problem enna nu purinjukitten.
Apdi pannunaa solution kandupudikkirathu easy.
So next time...
Exam aa irundhaalum, argument aa irundhaalum, confusion aa
irundhaalum...
First calm aagunga
Appuram think pannunga
Seriyaa?
Yennaaa...
Calm mind dhaan yeppavume namma super power.
Adutha adventure-la sandhippom!”
